# Timer Terminal User Interface (TUI) Application

This is a simple timer application built using the Bubble Tea framework V2 in Go. It allows users to input a number of seconds and starts a countdown timer. The application also handles user input for starting the timer and correcting it using the backspace key.

## Features
- Input a number of seconds to start the timer.
- Countdown display of the remaining time.
- User-friendly interface for starting and correcting input.
- 
## Usage
1. Run the application using `go run main.go`.
2. Type the number of seconds you want to set for the timer.
3. Press the "Enter" key to start the timer.
4. If you need to correct your input, use the "Backspace" key to delete characters.
5. The timer will count down and display the remaining time until it reaches zero.
